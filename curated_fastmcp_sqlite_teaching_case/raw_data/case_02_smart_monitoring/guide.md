# 案例 02｜夜間漏氣與能耗浪費

## 你會練到什麼
這一題會把監控數據直接轉成成本語言。  
你不只會看到夜間漏氣，還會把它換算成漏氣量、耗電量與年度浪費電費。

## 情境
工廠白天正常生產，但夜間停機後，空壓系統流量仍持續存在。  
請從 `data_case_02.csv` 找出非生產時段的異常流量，並估算其成本。

## 建議任務
```text
請讀取 data_case_02.csv，使用 Python（pandas + matplotlib）完成以下任務：
1. 找出 production_flag = 0 的非生產時段。
2. 計算非生產時段的平均 flow_m3_min 與平均 power_kw。
3. 假設這些流量都視為漏氣，計算：
   - 每日漏氣量（m3/day）
   - 每日浪費電量（kWh/day），請用 0.11 kWh/m3
   - 每年浪費電費（TWD/year），請用電價 3.2 元/kWh
4. 畫出 flow 與 power 趨勢圖，並標出非生產時段。
5. 用繁體中文輸出：
   - 是否存在夜間漏氣
   - 主要 KPI
   - 最可能現場檢查點
```

## 你可以檢查的答案
- 非生產時段長度：`12 小時`
- 非生產平均流量：`1.425 m3/min`
- 非生產平均功率：`18.77 kW`
- 每日漏氣量：`1026.0 m3/day`
- 每日浪費電量：`112.86 kWh/day`
- 年度浪費電費：`NT$ 131820.5`

## 延伸練習
1. 把結果改寫成 3 句管理摘要。
2. 將電價改成 `4.0 元/kWh` 後重新計算。
3. 加上門檻線，當非生產時段 flow 超過 `0.8 m3/min` 時視為異常。
